# WebCop.my - Digital Company Chop Generator

A complete solution for generating digital company chops with BCL.my payment integration for HD downloads.

## Features

- **Free Stamp Generation**: Create basic company chops with watermark
- **HD Download (RM3)**: Purchase watermark-free high-quality stamps
- **Multiple Templates**: Round, Box, and Info stamp styles
- **Real-time Preview**: Instant preview as you type
- **BCL.my Integration**: Secure payment processing with multiple methods
- **Responsive Design**: Works on all devices

## Payment Methods Supported

1. **FPX** (Online Banking)
2. **Credit/Debit Cards**
3. **Touch 'n Go eWallet**
4. **GrabPay**
5. **Boost**
6. **All Methods** (Let user choose)
